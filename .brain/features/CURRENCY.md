# Currency

The lounge shop uses **stardust** as its currency. Stardust represents a
lightweight progression resource for purchasing cosmetic or collectible items.

Shop listings and item variants are documented in `ITEMS.md`. Each catalog entry
has a small value tier that maps to its Stardust price. The initial shop UI still
displays a placeholder balance of `0`. Currency earning, persistence, and item
purchasing are not implemented yet; future progression systems should treat
Stardust as shared game state rather than dialogue state.
